    Welcome to Simba¶

    Modern API Testing Made Simple

    What is Simba?¶

    Simba is a powerful, modern API testing tool designed to streamline your API development and testing workflow. Similar to Postman but with enhanced features for GraphQL, WebSocket, and collaborative testing.

    Key Features¶

    • Support for all HTTP methods (GET, POST, PUT, PATCH, DELETE)
    • Rich request body editors (JSON, form-data, XML)
    • Headers and query parameter management
    • Multiple authentication methods
    • Native GraphQL query editor with syntax highlighting
    • Schema introspection and exploration
    • Variables and operation management
    • Auto-complete for GraphQL queries
    • Pre-request and test scripts (JavaScript)
    • Collection runner for automated testing
    • Data-driven testing with CSV/JSON files
    • Environment and global variables
    • Workspace management
    • Team permissions and sharing
    • Import/Export collections
    • Request history tracking

    System Requirements¶

    • Operating System: Windows 10+, macOS 10.15+, Linux (Ubuntu 20.04+)
    • Node.js: 20.x or higher (for local development)
    • Database: PostgreSQL 14+ and MongoDB 6+
    • Browser: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
